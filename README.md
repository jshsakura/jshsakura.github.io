# devterminal

터미널 스타일의 개인 포트폴리오 사이트.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion

## Features

- 터미널 명령어 기반 인터랙션 (`help`, `whoami`, `skills`, `career` 등)
- 19가지 터미널 테마 지원 (Catppuccin, Dracula, Nord, Tokyo Night 등)
- Tab 자동완성, 방향키 히스토리
- 숨겨진 이스터에그들

## Dev

```bash
npm install
npm run dev
```

## Deploy

GitHub Pages로 배포.

```bash
npm run build
```
